Roblox Job ID Joiner

Enter a Roblox Job ID to join a specific game server

How to find Job ID

  1. Open Roblox in a browser
  2. Join any game
  3. Look at the URL: ...?jobId=YOUR_JOB_ID
  4. Copy the Job ID after jobId=

Quick Join

Direct URL format:

https://your-app.vercel.app?job=JOB_ID

Example: https://your-app.vercel.app?job=abc123def
